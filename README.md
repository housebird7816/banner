# 引用：
#### AndroidStudio gradle:
- project的builde.gradle中：
repositories {
        maven {
            url 'https://dl.bintray.com/housebird/banner'
        }
    }
- app的builde.gradle中：
  implementation 'cn.housebird:banner:1.0.3'
  
#### Eclipse:
- 切换Android吧。
